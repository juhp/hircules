Hircules - an IRC client in haskell

Hircules (pronounced like the Greek hero) is an IRC client written in Haskell. The code is initially based on lambdabot and uses gtk2hs for the UI. It currently supports tabs for channels and private dialogs, but it is still quite simple. I'm planning to add more features like completions, CTCP extension support, coloring of nicks and conversations, better keybindings and UI, etc (see the TODO file for a longer list).

Releases

2003-05-15 Hircules 0.1 released (first release)

  • source tarball [84kB]
  • source rpm [105kB] (includes lambdabot 1.0 source too)
  • i386 rpm [556kB] (built on Red Hat Linux 9)
Features include:
  • tabs for channels and private dialogs
  • raw irc tab
  • all channel tab
  • lambdabot extension modules can be used, but most of them are disabled by default currently
Hopefully there will be a cvs repository up soon for better public access.

Building

To build Hircules you need:

  • ghc, the Glasgow Haskell Compiler (I'm using 5.04.3)
  • gtk2hs, a GTK 2 Binding for Haskell (you probably need cvs HEAD, alternatively you can download one of the following:)
    • gtk2hs source [399kB] (cvs snapshot with a few patches)
    • gtk2hs srpm [405kB]
    • gtk2hs i386 rpm [1.6MB] (built with ghc-5.04.3 on Red Hat Linux 9)
  • lambdabot 1.0
